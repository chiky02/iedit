📊 MODELOS DE BASE DE DATOS (Prisma):

1. Usuario (admin)
   - id: String (CUID)
   - email: String (UNIQUE)
   - password: String (bcryptjs hasheada)
   - nombre: String
   - rol: String (default: admin)
   - createdAt, updatedAt: DateTime

2. Ingreso
   - id: String (CUID)
   - monto: Decimal(12,2)
   - descripcion: Text
   - categoria: String
   - fecha: DateTime
   - createdAt, updatedAt: DateTime

3. Gasto
   - id: String (CUID)
   - monto: Decimal(12,2)
   - descripcion: Text
   - categoria: String
   - fecha: DateTime
   - createdAt, updatedAt: DateTime
